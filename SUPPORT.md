# Support

Thanks for using Clean Markdown Reader. This document outlines how to get help and report issues.

## Getting Help

### Documentation

- **README**: Start with the [README](README.md) for installation, features, and configuration options
- **Settings**: All extension settings are documented in VS Code under Settings > Clean Markdown Reader

### Common Issues

**Extension not activating?**
- Ensure VS Code version 1.108.0 or higher
- Try reloading VS Code (`Cmd+Shift+P` / `Ctrl+Shift+P` > "Developer: Reload Window")

**Files not appearing in the tree?**
- Check your filter pattern matches your file extensions
- Verify files aren't in excluded folders (`node_modules`, `.git`, etc.)

**Mermaid diagrams not rendering?**
- Ensure `clean-markdown-reader.renderMermaid` is enabled in settings
- Check diagram syntax at [mermaid.js.org](https://mermaid.js.org/)

## Reporting Issues

Before opening an issue:

1. Search [existing issues](https://github.com/amitsaddi/clean-markdown-reader/issues) to avoid duplicates
2. Update to the latest extension version
3. Test with extensions disabled to rule out conflicts

When reporting:

- VS Code version
- Extension version
- Operating system
- Steps to reproduce
- Expected vs actual behavior
- Relevant error messages (from Help > Toggle Developer Tools > Console)

[Open an issue](https://github.com/amitsaddi/clean-markdown-reader/issues/new)
